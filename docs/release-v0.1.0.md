# Port Sentinel v0.1.0

Initial public release of Port Sentinel.

## Highlights

- Inspect local listening ports and the processes that own them
- Search by PID, process name, port, or local address
- Kill a process directly by PID or by the port it is listening on
- Review and stop active Docker containers from the same dashboard
- Use the interface in multiple languages

## Notes

- Full socket inspection and process termination are currently Windows-first
- The project is intended for local development and machine administration
