## Repository Description

Windows-first local dashboard for inspecting listening ports, process owners, and Docker containers from one React UI.

### Title

`Port Sentinel v0.1.0`

### Release Notes

Initial public release of Port Sentinel.

Highlights:

- Monitor local listening ports and owning processes from one dashboard
- Search by PID, process name, port, or address
- Kill a process by PID or by the port it is listening on
- Review and stop active Docker containers
- Use the app in multiple interface languages

Important notes:

- Full socket and process actions are currently Windows-first
- The project is intended for local development and machine administration
